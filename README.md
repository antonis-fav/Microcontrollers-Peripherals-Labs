# Microcontrollers-Peripherals-Labs

Aristotle University 2022

This git contains all three labarotary assignments of the course **"Microcontrollers and Peripherals"**. All assignments are developed and designed to be executed on the development platform **STM32 NUCLEO-F401RE**. 

You can find extend documentation about this board here :https://www.st.com/resource/en/user_manual/um1724-stm32-nucleo64-boards-mb1136-stmicroelectronics.pdf.

Inside each folder, you will find the assignment's description in PDF format, along side with the corresponding source code and all the required Drivers files (.h, .c). I have uploaded the whole projects as they have been created inside **Keil MDK**, so that everyone can easily execute and replicate the desired results. 

You can download and find extended information regarding Keil MDK here: https://developer.arm.com/Tools%20and%20Software/Keil%20MDK.
