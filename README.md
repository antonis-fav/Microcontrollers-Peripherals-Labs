This repository contains all three laboratory assignments for the course **"Microcontrollers and Peripherals"**.  
All assignments are designed and developed to run on the **STM32 NUCLEO-F401RE** development board.

## Requirements
- **Hardware**: [STM32 NUCLEO-F401RE](https://www.st.com/resource/en/user_manual/um1724-stm32-nucleo64-boards-mb1136-stmicroelectronics.pdf)  
- **IDE/Toolchain**: [Keil MDK](https://developer.arm.com/Tools%20and%20Software/Keil%20MDK)  

## Repository Structure
Each folder includes:
- Assignment description in **PDF format**
- Corresponding **source code** (main.c)
- Required **Driver files(.h, .c)**
- Complete Keil MDK project for easy setup and replication of the desired results

## Usage
1. Clone this repository.  
2. Open the desired project in **Keil MDK**.  
3. Compile and flash it to the **STM32 NUCLEO-F401RE** board.
